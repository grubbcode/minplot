
To install with pip:   
    pip install svgdatashapes
    pip install svgdatashapes_dt   (if you need date/time support)

Or if you prefer, you can just download directly and place into your python source directory:
    svgdatashapes.py
    svgdatashapes_dt.py   (if you need date/time support) 


Then, in your python code do:
    import svgdatashapes

If you need date/time support also do:
    import svgdatashapes_dt   


svgdatashapes has been tested on python 2.7 and 3.6
